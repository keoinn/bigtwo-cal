# 大老二計分

4 人玩大老二、1 人記錄的手機版計分應用。

## 計分規則

- **一般局**：一人獨贏，三位輸家依剩餘手牌由少到多分別付 **100 / 200 / 300**
- **超過門檻**：任一位輸家剩餘手牌超過設定張數，改為 **200 / 400 / 600**，贏家扣 100，記錄者獲 100
- 分數與牌局紀錄皆**累計**儲存於瀏覽器

## 開發

```bash
npm install
npm run dev
```

## 建置

```bash
npm run build
npm run preview
```

## 功能

| 分頁 | 說明 |
|------|------|
| 記分 | 選贏家、輸入各玩家剩餘張數、預覽後確認 |
| 排行 | 累計分數排行榜 |
| 紀錄 | 歷史牌局，可刪除並回退分數 |
| 設定 | 門檻張數、玩家名稱、清零 |
